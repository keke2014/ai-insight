# AI洞察网站内容更新说明文档

## 一、 总述：自动化内容更新管线

AI 洞察网站的内容更新告别了传统的手工编写网页模式，而是采用了一套高度自动化的 **“数据驱动 + 脚本生成 + 一键部署”** 的工程化更新管线。

核心更新流程分为四个阶段：**数据准备 $\rightarrow$ 页面组装 (含热搜词库构建) $\rightarrow$ 质量门禁 $\rightarrow$ 自动化部署**。复杂的排版和发布逻辑均由 `scripts/` 目录下的系列脚本接管，内容创作者只需负责提供原始的、结构化的数据文件。

---

## 二、 分述：核心更新环节详解

### 1. 数据准备与调度 (Data Preparation & Orchestration)
网站的每日洞察内容彻底剥离了样式表现，直接以标准化的 JSON 格式进行存储。
- **结构化产出**：大模型（或创作者）每天会生成一份诸如 `data/daily-content-YYYY-MM-DD.json` 的纯数据文件。该文件包含了当天的资讯、摘要与分析，代码行数通常极少，极大地节约了编写的时间。
- **如何生成当天的 JSON (具体操作)**：
  - **默认生成**：如果是全新创作一天，运行 `python scripts/gen_daily_json.py`。会自动帮你在 `data/` 下创建带有今日日期的标准空白配置。
  - **指定日期生成**：如果您计划为任意未来的一天起草，可带上特定日期以及 `--template`，如：`python scripts/gen_daily_json.py 2026-03-22 --template`
  - **继承复刻**：如果您想要复用昨天的结构，可以使用 `--from` 快捷继承命令，如：`python scripts/gen_daily_json.py 2026-03-22 --from 2026-03-21`。
- **流程编排**：由 `scripts/ai_daily_orchestrator.py` 提供支持日志，它像一个统筹分配系统一样，管理、记录每一次生成任务的上下文与进度状态快照。

### 2. 页面组装与搜索提能 (HTML Generation & Global Search)
每天的内容准备完毕后，再由专门的脚本为其穿上“网页外衣”并提取全站语义：
- **组合生成**：使用 `scripts/gen_daily_html.py <日期>` 命令。该脚本自动读取当天的 JSON，并与 `templates/` 目下的 CSS / JS 设计模板进行组合，最后输出成完整的长篇 HTML 网页并保存到 `01-daily-reports/`。
- **全站搜罗库打底 (最新升级)**：在此阶段，系统会自动触发 `scripts/build_search_index.py`，全量扫描过往的所有的历史 JSON 库点，抽取出所有的标题和深度内容，输出成轻量级的 `search_index.json` 高性能热查图卡配置文件，这使得整个静态网页能拥有平滑实时的前台搜索功能！
- **首页自动联动**：除了生成子页面，脚本还会在首页 `index.html` 的最新进展面板和浏览日历上，动态插入新文档的超链接和内容提要。

### 3. 上线前质检门禁 (Quality Gate)
在推送到线上产生实际影响前，系统配置了严格的校验来防御低级失误：
- **安全与合规扫描**：`scripts/daily_quality_gate.py` 会在提交前被触发，强制检查生成的网页是否有失效链接、内部敏感词泄露或者是关键的 JSON 字段缺失（比如“明日关注”栏目是否意外为空）。任何 `Hard Fail` 级别的校验不通过都会被直接拦截。

### 4. 自动化分发与部署 (Automated Deployment & Sync)
一切完备后，系统通过主令脚本进行一站式分发：
- **底盘一键部署**：除了上述的微操指令外，创作者可以直接启动 `scripts/ai_daily_orchestrator.py finalize`，它会多线程齐下地包揽从内容推断到提交发布的全任务。
- **公私域版本控制**：为了兼顾内外场景需求，脚本还会同时调用 `scripts/sync_to_public.py`，将含私密内容的报告一键脱敏，同步生成到 `public/` 目录下用于外部分享。

## 三、 总结规范：如何用“极简发布脚本”完成全套更新流？

如果您是这套系统的维护者，**您不必单独记叙上述的所有分离终端命令**。
我们已经为您打包好了全流程的超级极简串并行一发方案：

只需在系统直接运行：
```bash
./scripts/one_click_publish.sh
```

**一键脚本执行流**：
1. **[步骤一 - 起草]** 脚本会自动跑起模板生成逻辑，若发现今日内容为空白则会自动排版今天的空表单。
2. **[步骤二 - 录入]** 脚本将**自动休眠中止**等待。您趁这个时候前去 `data/` 把当天的内容 JSON 补齐，补齐保存后回终端按个回车即可。
3. **[步骤三 - 自动化成型]** 剩下的事全部交给脚本流！它会串行扫描链接有效性并触发防失效门禁，质检成功后通过并发执行流同时生成包含最终页面组装、热查搜索词库（`search_index.json`）、生成对应的内外部隔离版本的网站配置，并一键推向 Github Pages 实现免停机的云更新。过程通常只需要几十秒钟！
